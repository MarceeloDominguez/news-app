## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/news-app.git
```

Go to the project directory

```bash
  cd news-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Start the tests

```bash
  npx jest --watchAll
```
